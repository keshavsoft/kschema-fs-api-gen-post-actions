KSchema API Generator

Generate Express.js API boilerplate instantly using reusable CLI actions, dynamic command loading, and JSON-driven architecture.

npx @keshavsoft/kschema-api-gen ShowAll

Get Started

Features

⚡ Fast API Generation

Create folders, controllers, routes, and boilerplate instantly.

🧠 JSON Driven

Single source of truth using actions.json and dynamic imports.

🚀 Scalable Architecture

Add new actions without modifying core resolver logic.

📦 Dynamic Loading

Commands are loaded dynamically with zero hardcoded maps.

Quick Start

npm install -g @keshavsoft/kschema-api-gen npx @keshavsoft/kschema-api-gen ShowAll

Architecture Flow

CLI ↓ parseInput.js ↓ resolveCommand.js ↓ loadCommand.js ↓ actions.json ↓ dynamic import() ↓ tasks/actions/ ↓ template generation

Example actions.json

[ { "cmd": "showAll", "file": "showAll", "exportFile": "ShowAll", "group": "GetMethods", "description": "Generate ShowAll GET action" } ]

Supported Commands

ShowAll

Generate GET all records action.

Insert

Generate POST insert action.

Distinct

Generate distinct records action.
